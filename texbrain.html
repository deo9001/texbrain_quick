    What this standalone mode does well

    • Edits `.tex` locally with autosave.
    • Renders common math such as inline math, `equation`, `equation*`, `align`, `align*`, `$$...$$`, and `\[...\]`.
    • Recognizes `\title`, `\author`, `\date`, `\maketitle`, sections, subsections, paragraphs, and lists.
    • Keeps unsupported content visible through readable fallback panels instead of failing silently.

    Known limits

    • It does not run a full TeX compiler, so TikZ, `tabularx`, figures, listings, and complex packages are shown as source fallbacks.
    • If the MathJax CDN is blocked, the editor still works, but math remains visible as TeX source.
    • This is separate from the full Svelte app, which provides PDF compilation, git workflows, multi-file projects, and collaboration.

    Accessibility notes

    The interface supports keyboard navigation, responsive layouts, visible focus states, and semantic labels. Use the skip link at the top of the page to jump directly into the editor.
